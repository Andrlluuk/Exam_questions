Правила составления документа:

Для обозначения сложности вопроса, нужно перед каждым вопросом поставить "3." или "4." или "5." Для обозначения обязательного вопроса (и его желаемой частоты появления в билетах) необходимо после сложности поставить еще одну цифру (которая обозначает частоту, например, если частота 2, то этот вопрос будет появляться в каждом втором билете) со знаком "." (например "3.2.") Также вопросы можно разбить по главам, тогда в каждом билете вопросы будут равномерно браться из разных глав. Для этого нужно в отдельной строке обозначить начало главы словом "Глава"
